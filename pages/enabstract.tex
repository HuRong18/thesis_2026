With the rapid development of IoT and AI technologies, computation-intensive and latency-sensitive applications have posed significant challenges to the computational capabilities at the network edge. Distributed edge computing effectively alleviates single-node bottlenecks through parallel processing across multiple nodes. To address the "straggler effect" common in distributed systems, coded edge computing introduces computational redundancy, allowing the recovery of original tasks from a subset of nodes, thereby significantly reducing computation latency. However, existing research on coded edge computing mostly treats downlink transmission as an independent multi-user communication problem, overlooking the strong correlation among computation results introduced by task encoding. If effectively exploited, this correlation offers tremendous potential for further data compression and latency reduction. Addressing these issues, this thesis introduces Distributed Source Coding (DSC) theory to investigate the mechanism of enhancing downlink transmission efficiency via result correlation. Specifically, aiming at two typical scenarios—full data reconstruction and linear function aggregation—this thesis proposes efficient transmission methods for coded computation results assisted by DSC. The main contributions are as follows:

(1) For general scenarios requiring full reconstruction of computation results, a DSC-assisted downlink cooperative transmission mechanism based on syndrome puncturing is proposed. Leveraging the source correlation revealed by the Slepian-Wolf theorem, a practical syndrome puncturing coding scheme based on LDPC codes is designed. By exploiting the algebraic structure introduced by coded computing, this scheme allows edge nodes to adaptively adjust compression rates according to their channel conditions. This enables flexible reallocation of transmission loads, effectively preventing nodes with poor channel conditions from becoming transmission bottlenecks. Furthermore, a downlink transmission latency minimization model is established. To solve this non-convex problem, a Convex-Concave Procedure (CCCP) based algorithm is developed to jointly optimize source compression rates and transmit power. Simulation results demonstrate that the proposed scheme significantly reduces downlink transmission latency compared to traditional independent transmission schemes.

(2) For scenarios requiring only the recovery of linear combinations of results (e.g., gradient aggregation in distributed machine learning), a DSC transmission method oriented towards linear aggregation is proposed. Since recovering individual sub-task results is unnecessary, the classic Slepian-Wolf bound is suboptimal. This thesis introduces nested linear codes from algebraic network information theory to construct a transmission model tailored for function computation by utilizing the linear structure of data. By designing nested codebooks with homomorphic properties, the central node can recover the target function value directly from the superimposed signals using simultaneous joint typicality decoding. Theoretical analysis proves that this method achieves a larger achievable compression region compared to full reconstruction. Addressing the complex boundary constraints of this region, a simplified optimization strategy is designed to further reduce communication overhead while ensuring computation accuracy.

In summary, by mining the intrinsic correlation among tasks in coded edge computing and deeply integrating distributed source coding with coded computing, this thesis provides a new perspective and effective solutions for resolving communication bottlenecks in edge computing systems.
